## 소개

무작위 로또 번호를 생성하고, 최근 10건의 로또 당첨 소식을 볼 수 있는 프로젝트
지도를 통해 근처의 복권 판매점도 확인하고 로또 용지의 QR 코드를 인식해 당첨 확인도 할 수 있습니다!
